
## インストール
```
sudo apt-get install software-properties-common
sudo add-apt-repository -y ppa:ethereum/ethereum
sudo apt-get update
sudo apt-get install ethereum
```

## genesisファイル作成
`nano genesis.json`

```
{
    "timestamp": "0x00", 
    "gasLimit": "2100000",
    "difficulty": "0",
    "config": {
        "chainId": 15,
        "byzantiumBlock": 0,
        "eip155Block": 0,
        "eip158Block": 0
    },
    "alloc": {}
}

```

## データベース作成
`geth init genesis.json`


## gethを走らせる
`nohup geth —networkid 15 &`


## コンソールに入る
`geth attach ipc://home/ubuntu/.ethereum/geth.ipc`


## アカウント作成
```
> personal.newAccount()
```

## マイニング開始
```
> miner.start()
```

## peer追加 *1つのノードから
```
> admin.addPeer()
```

## マイニング終了
```
> miner.stop()
```
